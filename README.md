# Bombeck Consulting

One-page website for Bombeck Consulting LLC — strategic advisory for Web3 and frontier tech in Latin America.

## Stack
- Plain HTML + CSS + vanilla JS
- Netlify Forms for contact form (no backend needed)

## Deploy
1. Push to GitHub (`fortunatagithub/bombeck-consulting`)
2. Connect repo to Netlify
3. Publish directory: `.` (root), no build command
4. Site name: `bombeck` → `bombeck.netlify.app`
